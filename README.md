# flask-test-app

A simple user CRUD manager built with Flask and PostgreSQL.

Live Demo: [View on Render](https://flask-test-app-8s5k.onrender.com)

## Features

* Create, read, update, and delete users
* User list filtering with a search query
* Form data validation and flash messages

## Useful commands

```bash
# Start Flask development server
make start

# Initialize database using .env variables
make build
